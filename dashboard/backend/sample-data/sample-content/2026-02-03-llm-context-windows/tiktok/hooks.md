# TikTok Hooks: LLM Context Windows

### Hook 1: "I fed an entire codebase into one prompt. Here's what happened."
**Type:** Curiosity gap
**Opening visual:** Screen recording of pasting massive text into Claude
**Why it works:** Creates irresistible curiosity about the outcome

### Hook 2: "Stop scrolling if you use AI for coding"
**Type:** Direct address
**Opening visual:** Face to camera, hand gesture to stop
**Why it works:** Calls out the target audience directly

### Hook 3: "Hot take: bigger context windows are a trap"
**Type:** Contrarian take
**Opening visual:** Bold text overlay on dark background
**Why it works:** Challenges the assumption that bigger = better

### Hook 4: "Nobody is talking about the attention bottleneck"
**Type:** Curiosity gap + authority
**Opening visual:** Diagram showing attention distribution
**Why it works:** Positions creator as having insider knowledge

### Hook 5: "I tested every context window size. The results surprised me."
**Type:** Surprising fact
**Opening visual:** Side-by-side comparison screenshots
**Why it works:** Promises data-backed revelation

**Ranking:** Hook 1 > Hook 3 > Hook 4 > Hook 2 > Hook 5
