# Instagram Caption Template

## Feed Post / Carousel Caption

```
{{HOOK_LINE — must grab attention in first 125 characters, appears above the fold}}

{{BODY — 3-5 short paragraphs or bullet points. Value-first. Each paragraph should be 1-2 sentences max.}}

{{CTA — Save this for later ✅ / Share with a dev friend / Drop a 🔥 if you agree / What's your take? Comment below}}

.
.
.
{{HASHTAGS — paste in first comment, not here}}
```

## Hashtag Block (First Comment)

```
{{5-7 BROAD HASHTAGS — 100k-1M posts}}
{{5-7 MID_RANGE HASHTAGS — 10k-100k posts}}
{{3-6 NICHE HASHTAGS — under 10k posts}}
```

## Carousel Guidelines

- **Slide 1 (Cover):** Bold statement or question. Large text. Brand colors. Must work as a thumbnail.
- **Slides 2-8 (Content):** One idea per slide. 30-50 words max. Visual hierarchy: header → supporting text → icon/graphic.
- **Slide 9 (Summary):** Quick recap of key points. "Save this carousel" CTA.
- **Slide 10 (CTA):** Follow for more, link to newsletter, or engagement prompt.

## Stories Guidelines

- **Frame 1:** Hook — same energy as carousel cover but vertical (9:16)
- **Frame 2-3:** Core content — keep text minimal, use stickers/GIFs for energy
- **Frame 4:** Interactive element — poll, quiz, or question sticker
- **Frame 5:** CTA — swipe up / link sticker / "See our latest post"

## Emoji Usage

- Use sparingly and intentionally — max 3-5 per caption
- Lead with emoji on bullet points for scannability
- Never use emoji mid-sentence (looks cluttered)
- Preferred: ✅ 🔑 💡 🚀 📌 🔗 🎯 ⚡
