# Thread Structure Template

## Thread Format

Each post must be under 500 characters. Number every post (1/N, 2/N...).

## Structure

### Post 1/N — THE HOOK
```
{{BOLD CLAIM or SURPRISING INSIGHT or PROVOCATIVE QUESTION}}

This changes how I think about {{TOPIC}}.

A thread 🧵👇
```
*Goal: Self-contained and shareable. If someone screenshots only this post, it should make them want to find the rest.*

### Posts 2-4/N — THE SETUP
```
{{CONTEXT — Why this matters. What most people get wrong. The conventional wisdom you're challenging.}}
```
*Goal: Establish the problem or gap in understanding. Build tension.*

### Post 5/N — ENGAGEMENT HOOK #1
```
{{QUESTION or MINI-POLL}}

Quick question: {{QUESTION THAT INVITES REPLIES}}
```
*Goal: Drive engagement. Get people into the replies early.*

### Posts 6-9/N — THE CORE ARGUMENT
```
{{EVIDENCE, EXAMPLES, DATA POINTS}}

Each post = one idea, one example, or one insight.
```
*Goal: Build the argument. Use concrete examples. Reference sources.*

### Post 10/N — ENGAGEMENT HOOK #2
```
{{CONTRARIAN ASIDE or NUANCE}}

To be fair: {{COUNTERARGUMENT YOU ACKNOWLEDGE}}

But here's the thing...
```
*Goal: Show intellectual honesty. Invite debate.*

### Posts 11-13/N — IMPLICATIONS
```
{{SO WHAT? What does this mean in practice? What should the reader DO differently?}}
```
*Goal: Make it actionable. Connect insight to the reader's daily work.*

### Post 14/N — THE TAKEAWAY
```
{{ONE-SENTENCE SUMMARY OF THE CORE INSIGHT}}

That's the key insight: {{RESTATE SIMPLY}}
```

### Post 15/N — THE CTA
```
If you found this useful:

♻️ Repost to help others see it
💬 What's YOUR experience with {{TOPIC}}?
📬 I write about this stuff weekly — follow for more

{{LINK TO NEWSLETTER or SUBSTACK if relevant}}
```

## Conversation Starters (Post as Replies)

Reply to your own thread to seed the discussion:

1. **Experience prompt:** "Curious — has anyone tried {{SPECIFIC APPROACH}}? What happened?"
2. **Devil's advocate:** "The strongest counterargument I've heard is {{COUNTER}}. What do you think?"
3. **Tool/resource share:** "If you want to try this yourself, {{RESOURCE}} is a great starting point."
4. **Hot take follow-up:** "Spicy follow-up: I think {{RELATED HOT TAKE}}. Fight me in the replies."

## Rules

- Every post must stand alone — shareable as a screenshot
- No post should require reading the previous post to make sense
- Engagement hooks at posts 5 and 10 (adjustable)
- Use line breaks for readability (no wall-of-text posts)
- Emojis: use at start of posts and for visual breaks, not mid-sentence
